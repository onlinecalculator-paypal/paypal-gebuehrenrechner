## ⚙️ Wie funktioniert die Berechnung?

Die PayPal Gebühr basiert auf einer einfachen Formel:

Betrag × Prozentsatz + Fixgebühr

**Beispiel:**
- 100€ × 2,49% = 2,49€
- + 0,35€ Fixgebühr  
→ Gesamtgebühr: 2,84€
